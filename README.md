# Optimism Block Tools

A simple collection of manual tools and methods for working with Optimism blocks without scripts or coding.  
This repository documents:
- How to retrieve Optimism block data via RPC
- How to compute RLP values manually
- How to decode fields like `baseFeePerGas`, `extraData`, `gasUsed`, etc.
- Tips for solving CTF challenges related to block structure and transaction analysis

The repo is intended for beginners who want to work with raw block data step-by-step without writing code.

More tools and notes will be added as I continue learning.
